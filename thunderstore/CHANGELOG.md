# Changelog

## 1.0.0

- Initial release
- WebSocket event server on ws://localhost:8765/events
- Player events: death, damage, healed, joined, left, emote
- Round events: start, end, ship landing/leaving, day/moon change, quota, revive
- Enemy events: spawned, killed, state changed, hit
- Monster-specific events for 14 enemy types
- Item events: grabbed, dropped, apparatus pulled
- Facility events: landmine, turret mode, teleporter
- Player filtering (local player only, player name filter)
- Designed for Streamer.bot integration
